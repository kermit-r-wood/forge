# Forge

[English](README_EN.md) | [中文](README.md)

## 简介

Forge 是一个 FDM 打印机多色生成工具。它利用普通线材的半透明特性，通过叠加不同颜色来混合出新色彩。算法会计算如何分层才能用你手头的线材还原目标图片。

![Introduction](doc/intro_cn.png)

## 特性

*   **本地运行**：基于 PySide 开发的桌面应用，直接在电脑上运行。
*   **零配置**：下载即可使用，无需搭建服务器或复杂安装。

## 使用说明

1.  **拖入图片**：直接把图片拖进软件窗口。
2.  **设置参数**：设定尺寸、层高，并选择你拥有的线材颜色。
3.  **生成模型**：点击生成。软件会计算分层方案并保存为 3MF 文件。
4.  **切片打印**：用切片软件打开 3MF 文件并打印。

## 原理简述

PLA 和 PETG 等常见线材都有点透光。不同颜色叠加会产生混合效果：比如红色层盖在白色层上面可能偏粉。Forge 模拟这种光线透射效果，计算出如何利用现有线材层叠出最接近原图的效果。
